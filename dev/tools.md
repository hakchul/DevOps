# Tools

## Windows Terminal

* keybindins (profiles.json)

    ```json
    {
        "keybindings": [
            { "command": "newTab",   "keys": ["ctrl+t"] },
            { "command": "closeTab", "keys": ["ctrl+w"] },
            { "command": "prevTab",  "keys": ["ctrl+pgup"] },
            { "command": "nextTab",  "keys": ["ctrl+pgdn"] }
        ]
    }
    ```
